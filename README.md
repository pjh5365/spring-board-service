# spring-board-service

스프링부트로 만들어진 게시판 서비스 프로젝트

뷰는 PHP 로 만들었었던 파일을 가져와 현재 프로젝트에 맞게 수정하여 적용하였다.

[PHP 홈페이지 소스코드](https://github.com/pjh5365/ServerManagementPage)

## 개발 환경

- Intellij IDEA Ultimate 2022.23
- Spring Boot 2.7.14

- JAVA 11 (Temurin)
- Gradle 8.2.1

## 세부 기술

#### Spring Boot

- Spring Web
- MariaDB Driver
- Spring Data JPA
- Thymeleaf
- Spring Security
- Lombok
